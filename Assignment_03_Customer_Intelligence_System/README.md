# Assignment 03 – Customer Intelligence System

## Overview

This assignment was completed as part of the **Celebal Excellence Internship (CEI) 2026** under the **Data Science (DS001)** track at **Celebal Technologies**.

The objective of this assignment was to develop an end-to-end Customer Intelligence System using supervised and unsupervised machine learning techniques. The project focuses on customer classification, ensemble learning, and clustering to generate predictive insights and actionable customer segmentation.

---

## Problem Statement

**Develop a Customer Intelligence System using classification, ensemble learning, and clustering.**

The assignment involved designing and implementing a complete machine learning workflow covering:

- Data Preprocessing
- Exploratory Data Analysis (EDA)
- Feature Engineering
- Classification Models
- Ensemble Learning
- Customer Segmentation
- Clustering Analysis
- Model Evaluation
